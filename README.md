# WorldCup Chatbot
Chatbot baseado em vue que tira-dúvidas acerca
da Copa do Mundo de seleções FIFA 2026


## Features

### Frontend features
1. Controle de sessão de usuário
2. Envio e atualização de mensagens sob contexto

### Backend features
1. Suporte à modelos auto-hospedados do Ollama
2. Estrutura da REST API de acordo com OpenAI Like contract
3. Gerenciamento de janela de contexto (tamanho da Sliding Window)

## Stack
- Vue.js
- Tailwind
- PostgreSQL
- Python e frameworks relacionados: FastAPI, SQLAlchemy

## Requirements
Para instalar os requisitos para execução do **Backend** com o gerenciador e pacotes **Poetry** rode:
``` poetry install ```
Para instalar os requisitos para execução do **Frontend** rode:
``` npm install ``
